Setup & CDN

To use jQuery, include this script in your head:

<script src="https://code.jquery.com/jquery-3.7.1.min.js"></script>

Selectors

jQuery uses CSS-style selectors to find elements.

  • $('p') selects all paragraphs.
  • $('#test') selects element with id="test".
  • $('.menu') selects elements with class="menu".
